Using the QGIHtml widget:

QGIHtml is a text/HTML widget for placing text directly onto a QBoard.

Tips:

  • Right-click to get the context menu.
  • Double-click to edit it in-line.
  • Middle-click to open an editor dialog, where you can edit it as HTML.

Be aware: if any of your edits make for invalid HTML, the changes will be lost when the editor is closed! So be careful!

Note that this widget supports only a subset of HTML, as described in the API section named 'Supported HTML Subset' in the Qt Assistant documentation. Also note that QTextEdit::setHtml() likes to mangle your HTML a bit, so don't be alarmed when it changes between edits.
