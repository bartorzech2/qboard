This is the LICENSE file for libS11nQt.

libS11nQt is based off of libs11n, which is Public Domain, and extends
libs11n with features from libqt. Since libqt has a variable license,
with different options depending on the version, libS11nQt allows the
user to select any license(s) which their version of Qt allows (whether
Open Source/Free Software or Commercial).
